## Functional Programming

- It is all about seperation of concerns
- seperation of data and (behavior of function) functions
- objects of fp are  immutable (can't be changed)
- Pillar of FP is pure function

```

// Amazon shopping

  active: true,
  cart: [],
  purchases: []
}
```
